## Context

Markdown is simple once it clicks, but the first encounter can feel abstract. This concept explored a more playful path: teach syntax through a mystery-style narrative where each formatting rule unlocks progress.

## Contribution

I shaped the learning flow, interface direction, and content structure for an experience that treats documentation skills as something interactive rather than dry reference material.

## What this demonstrates

- Ability to turn technical learning into approachable UX.
- Strong interest in developer education and documentation.
- Product thinking beyond CRUD screens: narrative, motivation, and user progress.
